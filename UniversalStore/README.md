# 🛍️ Универсальный магазин (Portable версия)

Полностью портативная информационная система магазина на **Python + PyQt6 + MySQL**.

Идеально подходит для **демонстрационного экзамена** — легко переносится с GitHub, быстро настраивается и универсальна под любую тематику.

## ✨ Особенности

- **Всё в одном `.py` файле** — максимальная переносимость
- Ролевая модель: **Гость → Клиент → Менеджер → Администратор**
- Поиск, фильтрация по поставщику, сортировка по остатку
- Карточки товаров с изображениями и подсветкой скидок/остатка
- Администратор: добавление, редактирование, удаление товаров + выбор фото
- Работа с заказами (просмотр + создание)
- Полностью универсальная структура БД — легко поменять под **книги, технику, одежду** и т.д.

## 🚀 Быстрый старт

### 1. Клонирование с GitHub
```bash
git clone https://github.com/ВАШ_ЛОГИН/UniversalStore.git
cd UniversalStore
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Создание базы данных
1. Запустите **MySQL** (Open Server, XAMPP, Docker и т.д.)
2. Откройте любой клиент MySQL (phpMyAdmin, DBeaver, HeidiSQL, MySQL Workbench)
3. Выполните скрипт **`database.sql`** целиком

### 4. Запуск программы
```bash
python main.py
```

**Тестовые данные для входа:**
| Роль            | Логин              | Пароль      |
|-----------------|--------------------|-------------|
| Администратор   | admin@shop.ru      | admin123    |
| Менеджер        | manager@shop.ru    | manager123  |
| Клиент          | client@shop.ru     | client123   |
| Гость           | —                  | Кнопка "Войти как гость" |

## 🔧 Как сделать под другую тему (книги, техника, одежда и т.д.)

### Шаг 1 — Измените базу данных
Отредактируйте файл `database.sql`:
- Измените `INSERT` в таблицы:
  - `Categories`
  - `Manufacturers`
  - `Suppliers`
  - `Products` (самое важное)
- Можно полностью заменить примеры товаров

### Шаг 2 — Измените название магазина
В файле `main.py` в самом верху найдите строку:
```python
SHOP_NAME = "Универсальный магазин товаров"
```
Замените на своё название, например:
```python
SHOP_NAME = "Книжный мир"
```

### Шаг 3 (опционально)
- Поменяйте цвета в переменных `COLOR_DISCOUNT` и `COLOR_OUT_OF_STOCK`
- Добавьте свои изображения в папку `images/`

Готово! Программа теперь работает под новую тематику.

## 📦 Сборка в .exe (Windows)

### Вариант 1 — Простая сборка (рекомендуется)
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "Магазин" main.py
```

Готовый файл появится в папке `dist/`

### Вариант 2 — С иконкой и всеми файлами
```bash
pyinstaller --onefile --windowed --icon=icon.ico --add-data "images;images" main.py
```

> **Примечание**: PyQt6 делает `.exe` довольно тяжёлым (~150-250 МБ). Это нормально.

### 1. Создайте репозиторий на GitHub
- Название: `UniversalStore` или `Магазин-товаров`
- **НЕ** добавляйте README при создании (мы добавим свой)

```bash

git commit -m "Initial commit: Универсальный магазин"
```

### 3. Привяжите удалённый репозиторий
```bash
git remote add origin https://github.com/ВАШ_ЛОГИН/UniversalStore.git
git branch -M main
git push -u origin main
```

### 4. Обновление в будущем
```bash
git add .
git commit -m "Обновление функционала"
git push
```

**Рекомендуемые файлы в `.gitignore`** (уже есть в проекте):
- `__pycache__/`
- `venv/`, `.venv/`

- `output/`
